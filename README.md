# Mildew Detection in Cherry Leaves

## Introduction

This project focuses on developing a machine learning model to distinguish healthy cherry leaves from those affected by powdery mildew. The goal is to create a system that quickly and accurately identifies affected leaves, aiding in early disease management and prevention.

## Prerequisites

This project comprises three main components: the Mildew Detection App, the Jupyter Notebooks for analysis and model development, and the Interactive Dashboard. Below are the prerequisites for each component.

### Mildew Detection App

To use the Mildew Detection App, the following are required:

- **Web Browser**: Modern web browsers like Chrome, Firefox, Safari, or Edge.
- **Internet Connection**: Needed for accessing the app and uploading cherry leaf images.
- **Image Formats**: JPEG, PNG or other common image formats supported for analysis.

### Jupyter Notebooks

For running and interacting with the Jupyter Notebooks, ensure the following setup:

- **Python 3.x**: Install Python 3.x from [Python.org](https://www.python.org/downloads/).
- **Jupyter Notebook**: Install Jupyter Notebook following instructions from [Jupyter.org](https://jupyter.org/install).
- **Python Libraries**: Install libraries listed in `requirements.txt`. Key libraries include TensorFlow, NumPy, Pandas, Matplotlib, Seaborn, and Keras.
- **Dataset**: Access to the cherry leaves dataset for model training and evaluation.

### Interactive Dashboard

Requirements for accessing and using the Dashboard:

- **Web Browser**: Compatible with modern web browsers for optimal viewing experience.
- **Internet Connection**: Necessary to access the dashboard and view analysis results and model metrics.

### Python Library Versions

The project has been developed with specific library versions to ensure compatibility. Install these libraries as per the `requirements.txt` file in the repository. Key libraries and their versions include:

- `numpy==1.19.2`
- `pandas==1.1.2`
- `matplotlib==3.3.1`
- `seaborn==0.11.0`
- `plotly==4.12.0`
- `streamlit==0.85.0`
- `scikit-learn==0.24.2`
- `tensorflow-cpu==2.6.0`
- `keras==2.6.0`
- `protobuf==3.20`
- `altair<5`

Run `pip install -r requirements.txt` in your project environment to install these dependencies.
